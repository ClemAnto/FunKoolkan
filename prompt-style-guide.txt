== NOTE D'USO ==

- Usare sempre "white background" per sprite personaggi (per ritaglio facile)
- Per il background usare "no characters in foreground" per evitare elementi indesiderati
- Se l'AI aggiunge testo indesiderato: aggiungere "no text, no letters, no numbers" al prompt
- Per coerenza di stile tra sessioni diverse: iniziare sempre con lo stesso seed/stile di riferimento
- Dopo il primo set approvato (Rana lv1-4): usare quell'immagine come "image prompt" o reference
  per mantenere lo stile coerente sulle specie successive
- Risoluzione target export: 256x256px (retina), poi ridimensionare a 128x128 per base
- Formato: PNG con sfondo trasparente (ritagliare dopo generazione)
